# TKD Trainer Frontend
Ionic frontend for TKD trainer platform


## Develop 

### Install
Make sure you have nodejs installed. NVM is recommended. Follow instrunctions at [NVM github page](https://github.com/nvm-sh/nvm)


Install ionic/cli and native compilers

`npm install -g @ionic/cli native-run cordova-res`


Run locally by running `ionic serve`
